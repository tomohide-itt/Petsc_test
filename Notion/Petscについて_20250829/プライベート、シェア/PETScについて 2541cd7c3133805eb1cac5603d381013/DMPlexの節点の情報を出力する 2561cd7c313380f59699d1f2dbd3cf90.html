DMPlexの節点の情報を出力する

節点情報を出力する関数

下記の関数を用意（006_Gmsh_DMPlex_WriteNode/gmsh3.cpp内）

static PetscErrorCode WriteNode( DM dm )
{
  PetscFunctionBeginUser;

  PetscMPIInt rank;
  MPI_Comm_rank( PETSC_COMM_WORLD, &rank );

  // DMPlex内部の節点IDの範囲とグローバル節点番号ISを取得
  PetscInt vStart, vEnd;
  IS vIS;
  const PetscInt *vnum;
  PetscCall( DMPlexGetDepthStratum( dm, 0, &vStart, &vEnd ) );
  PetscCall( DMPlexGetVertexNumbering( dm, &vIS ) );
  PetscCall( ISGetIndices( vIS, &vnum ) );

  // 座標アクセスの準備
  DM cdm;
  Vec coords;
  const PetscScalar *ca;
  PetscInt dim;
  PetscCall( DMGetCoordinateDM( dm, &cdm ) );
  PetscCall( DMGetCoordinatesLocal( dm, &coords ) );
  PetscCall( VecGetArrayRead( coords, &ca ) );
  PetscCall( DMGetCoordinateDim( dm, &dim ) );

  for( PetscInt p=vStart; p<vEnd; ++p )
  {
    // グローバル頂点番号（負値は-(id+1）が真のID）
    PetscInt g = vnum[p-vStart];
    PetscBool own = PETSC_TRUE;
    if( g < 0 )
    {
      g = -(g+1);
      own = PETSC_FALSE;
    }

    // 座標を安全に読み出し
    const PetscScalar *xp = NULL;
    PetscCall( DMPlexPointLocalRead( cdm, p, ca, &xp ) );
    if( !xp ) continue;

    // 出力
    printf( "%3d, %3d, %3d, %3d, ", (int)rank, (int)p, (int)g, (int)own );
    if( dim >= 1 ) printf( "%15.5e", (double)xp[0] );
    if( dim >= 2 ) printf( "%15.5e", (double)xp[1] );
    if( dim >= 3 ) printf( "%15.5e", (double)xp[2] );
    printf( "\n" );
  }
 
  PetscCall( ISRestoreIndices( vIS, &vnum ) );
  PetscCall( VecRestoreArrayRead( coords, &ca ) );
  PetscFunctionReturn( PETSC_SUCCESS );
}

この関数は，領域分割（分配）済みのDMPlexから，各ランクが持つ節点の一覧を取り出し，

rank, plex_point_ID, global_point_ID, owned, x, y, z を標準出力するもの．

  • plex_pointID： DMPlex内部の「点ID」
  • global_point_ID：PETScがつけたグローバル節点番号．ゴースト節点は，負値で渡されるので -(g+1) で本来のIDに直している．GmshのNodeTagではない．PETScが振りなおした番号であり，GmshのNodeTagが必要なら，分配前にラベルへ埋め込んで取得する必要がある．
  • owned：1=このランクの所有する節点，0=ゴースト節点（隣接ランクが所有する節点の複製）

以下，関数の説明

  • DMPlex内部の節点IDの範囲とグローバル節点番号ISを取得
    	// DMPlex内部の節点IDの範囲とグローバル節点番号ISを取得

  IS vIS;

  PetscCall( ISGetIndices( vIS, &vnum ) );
    • DMPlexGetDepthStratum( dm, 0, &vStart, &vEnd )
      • DMPlex の depth=0（頂点）のIDの範囲 [vStart, vEnd) を取得
    • DMPlexGetVertexNumbering( dm, &vIS ) → ISGetIndices( vIS, &vnum )
      • ローカル節点 p に対応するグローバル節点IDを，配列 vnum[p - vStart] に格納
      • ISは，PETScの「Index Set」（インデックス集合）．整数インデックスの集まりを表す軽量オブジェクトで，部分集合の指定・並べ替え・抽出ができる．
        const PetscInt *a;
ISGetIndices(is, &a);
// a[i] を読む（書き換えない）
… 
ISRestoreIndices(is, &a);  // ← 必ず対応して呼ぶ

        のように中身を取り出す．最後に必ずRestoreを忘れずに呼ぶ

  • 座標アクセスの準備

    • DMGetCoordinateDM( dm, &cdm )
      • 座標を管理するサブDMを取得
    • DMGetCoordinatesLocal( dm, &coords )
      • DMPlexからローカル節点の座標ベクトルを取得
    • VecGetArrayRead( coords, &ca )
      • 生配列ポインタを取得（変換?）
    • DMGetCoordinateDim( dm, &dim )
      • DMPlexから2D/3Dなどの埋込次元を取得
出力の並び・混ざり対策

MPI実行では， std::cout / printf をそのまま使うと，ランク間で行が混ざることがあるので，出力をきれいにするため，PETScの同期出力を使う

PetscSynchronizedPrintf(PETSC_COMM_WORLD, "%3d, ...\n", rank, ...);
// ループ後に
PetscSynchronizedFlush(PETSC_COMM_WORLD, PETSC_STDOUT);
