連立方程式を解く最小コード

PETScを使って，並列線形方程式ソルバーの簡単なサンプルコードを試す

1次元のラプラシアン型行列を作り，CG法（共役勾配法）で連立方程式 Ax=bAx=bAx=b﻿ を解く

コード
#include <petscksp.h>

int main(int argc,char **argv){
  PetscCall(PetscInitialize(&argc,&argv,NULL,NULL));

  PetscInt n = 50, rs, re;
  Mat A; Vec x,b; KSP ksp;

  PetscCall(MatCreate(PETSC_COMM_WORLD,&A));
  PetscCall(MatSetSizes(A,PETSC_DECIDE,PETSC_DECIDE,n,n));
  PetscCall(MatSetFromOptions(A));
  PetscCall(MatSetUp(A));
  PetscCall(MatGetOwnershipRange(A,&rs,&re));
  for (PetscInt i=rs;i<re;i++){
    if(i>0)   PetscCall(MatSetValue(A,i,i-1,-1.0,INSERT_VALUES));
    PetscCall(MatSetValue(A,i,i, 2.0,INSERT_VALUES));
    if(i<n-1) PetscCall(MatSetValue(A,i,i+1,-1.0,INSERT_VALUES));
  }
  PetscCall(MatAssemblyBegin(A,MAT_FINAL_ASSEMBLY));
  PetscCall(MatAssemblyEnd(A,MAT_FINAL_ASSEMBLY));

  PetscCall(VecCreate(PETSC_COMM_WORLD,&b));
  PetscCall(VecSetSizes(b,PETSC_DECIDE,n));
  PetscCall(VecSetFromOptions(b));
  PetscCall(VecDuplicate(b,&x));
  PetscCall(VecSet(b,1.0));

  PetscCall(KSPCreate(PETSC_COMM_WORLD,&ksp));
  PetscCall(KSPSetOperators(ksp,A,A));
  PetscCall(KSPSetType(ksp,KSPCG));
  PetscCall(KSPSetFromOptions(ksp));
  PetscCall(KSPSolve(ksp,b,x));

  PetscCall(KSPDestroy(&ksp));
  PetscCall(VecDestroy(&x));
  PetscCall(VecDestroy(&b));
  PetscCall(MatDestroy(&A));
  PetscCall(PetscFinalize());
  return 0;
}

ビルド

以下でビルドをする

mpicc poisson.c -o poisson $(pkg-config --cflags --libs petsc)

$(pkg-config --cflags --libs petsc) は，pkg-configというツールを使ってPETScをコンパイル・リンクするために必要なフラグを自動で取得する仕組み

  • pkg-config

    ライブラリのヘッダやリンクオプションを .pc ファイルにまとめて管理する仕組み

    C\C++プログラムをビルドするときに「どの -I （インクルードパス）や -L （ライブラリパス）や -I （リンクライブラリ）が必要か」を自動で取得する

    petsc.pc がPETScのインストールディレクトリに置かれている

  • --cflags と --libs

    --clfags ：コンパイルに必要なオプション（主に -I インクルードパスや -D マクロ定義）

    --libs ：リンクに必要なオプション（主に -L ライブラリパスと -l ライブラリ名）

    PETScをソースからビルドした場合，

    $ pkg-config --clfags petsc
-I/home/****/petsc/include -I/home/****/petsc/arch-linux-c-opt/include

$ pkg-config --libs petsc
-L/home/****/petsc/arch-linux-c-opt/lib -lpetsc -lmpi -lmpifort -lgfortran -lm -lstdc++

    となる．$(pkg-config --cflags --libs petsc) によって，自動的に正しいインクルードパス・ライブラリパスを拾ってくれる．（手作業で-I/home/…を書かなくてよい）

    ただし，システムに複数のPETScがあるとき，どの petsc.pc を使っているかが重要

    pkg-config --variable=prefix petsc

    で確認することができ，これが $PETSC_DIR/$PETSC_ARCH （例えば home/****/petsc/arch-linux-c-opt ）を指していればOK

    もし， /usr/lib/x8664-linux-gnu/pkgconfig/petsc.pc を指している場合は，パッケージ版のPETScを参照している．

実行

以下で実行する

mpiexec -n 2 ./poisson -ksp_monitor -ksp_converged_reason

実行すると以下のように出力される（3回の反復回数で収束）

  0 KSP Residual norm 4.449971910024e+02
  1 KSP Residual norm 3.373969169438e+01
  2 KSP Residual norm 4.068610088604e-12
  Linear solve converged due to CONVERGED_RTOL iterations 2

-ksp_monitor -ksp_converged_reason は，PETScの実行時オプションであり，ソルバの収束挙動を観察するためのデバッグ・可視化用オプション

  • -ksp_monitor

    Krylov Subspace Solver (KSP)の反復ごとの残差ノルムを表示する

  • -ksp_converged_reason

    ソルバ終了時に「なぜ止まったか」を表示
